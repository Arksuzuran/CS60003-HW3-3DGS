# 当前工程边界与限制

## 主线要求

- 物体 B 必须是严格意义上的文本到 3D 结果。
- 物体 C 必须是严格意义上的单图到 3D 结果。
- fallback 只能发生在实现路径内部，不能把任务退化成占位 mesh 或静态图片拼接。

## 预期风险

### 1. 权重不可达

集群无法稳定访问 Hugging Face 等站点，因此必须在本地预先准备权重。

### 2. 文本到 3D 显存和依赖波动

虽然集群显卡为 A100 40GB，但 `threestudio` 不同配置对扩展、CUDA 和权重路径很敏感，必须准备严格的路径校验与 fallback。

### 3. 报告编译

在线 TeX bundle 可能不可达，因此 PDF 编译不应只依赖集群。
